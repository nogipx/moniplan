# Licensify Telegram Bot

Телеграм бот для обработки запросов лицензий и генерации лицензионных файлов для приложений.

## Особенности

- Прием и обработка запросов лицензий (`.mlr` файлы)
- Генерация лицензионных файлов с криптографической защитой
- Валидация сигнатур и проверка срока действия запросов
- Интеграция с Dart-скриптами для переиспользования криптографической логики

## Установка

1. Установить зависимости Python:
```bash
pip install -r requirements.txt
```

2. Установить необходимые зависимости Dart (если нужно):
```bash
cd dart_scripts
dart pub get
```

3. Настроить конфигурацию:
- Создать файл `.env` с переменными окружения (на основе `.env.example`)
- Указать токен бота и путь к ключам шифрования

## Запуск

```bash
python -m bot.bot
```

## Структура проекта

- `bot/` - Исходный код Telegram бота
- `utils/` - Утилиты Python для работы с лицензиями
- `dart_scripts/` - Dart-скрипты для криптографических операций
