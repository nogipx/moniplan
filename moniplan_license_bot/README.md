# MoniPlan License Bot

Telegram-бот для продажи и управления лицензиями MoniPlan. Позволяет пользователям приобретать и активировать лицензии на неограниченном количестве устройств.

## Возможности

- Оформление бесплатной TRIAL лицензии на 2 месяца
- Покупка годовой PRO лицензии
- Активация лицензий на неограниченном количестве устройств
- Привязка лицензий к конкретному устройству
- Просмотр списка активированных устройств
- Интеграция с Telegram Bot Payments API

## Требования

- Python 3.8+
- aiogram 3.0+
- cryptography 39.0+
- SQLite 3

## Установка

1. Клонируйте репозиторий
2. Создайте виртуальное окружение и активируйте его:
   ```
   python -m venv venv
   source venv/bin/activate  # Linux/macOS
   venv\Scripts\activate     # Windows
   ```
3. Установите зависимости:
   ```
   pip install -r requirements.txt
   ```
4. Создайте файл `.env` на основе `.env.example`
5. Заполните необходимые переменные окружения в файле `.env`

## Использование

Запуск бота:

```bash
python bot.py
```

## Структура проекта

- `bot.py` - Основной файл бота, содержит обработчики команд и сообщений
- `models.py` - Модели данных (User, License, Device)
- `database.py` - Класс для работы с SQLite базой данных
- `license_service.py` - Сервис для работы с лицензиями
- `utils.py` - Вспомогательные функции и сообщения бота

## Процесс лицензирования

1. Пользователь приобретает доступ к функциям (TRIAL или PRO)
2. Пользователь активирует устройство через бота
3. Бот генерирует файл лицензии для устройства
4. Пользователь импортирует файл лицензии в приложение

## Настройка интеграции с Telegram Bot Payments API

1. Получите токен бота у [@BotFather](https://t.me/BotFather)
2. Настройте платежи через [@BotFather](https://t.me/BotFather) командой /mybots -> Выберите вашего бота -> Payments
3. Добавьте полученный PAYMENT_PROVIDER_TOKEN в .env файл
